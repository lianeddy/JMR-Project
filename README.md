# Jatimulya Regency Landing Page

This is a React 19 project built with Vite.

## Prerequisites

- Node.js (version 18 or higher recommended)
- npm (comes with Node.js)

## How to Run Locally

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Once the server starts, open the URL provided in the terminal (usually `http://localhost:5173`).

## Project Structure

- `src/App.jsx`: Main landing page component.
- `src/constants.ts`: Data constants used in the landing page.
- `src/main.jsx`: Entry point for React.
- `index.html`: Entry point for Vite.
